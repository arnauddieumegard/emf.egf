What's New in EGF 0.2.0

From EGF 0.1.0 to EGF 0.2.0:

  • Persistence of factory components and tasks in fcore files
  • Extensibility of the EGF structure (e.g., ability to enrich the default viewpoints with new viewpoints, new types)
  • Improvement of the EGF structure: support of output, input/output parameters for workflow definition, ability to execute factory components dynamically, user interface usability
  • Refactoring of patterns: ability to support new languages and engines, pattern injection
  • Improvement for patterns: user interface, pattern inheritance, pattern call
