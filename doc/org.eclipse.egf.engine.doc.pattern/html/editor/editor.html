Pattern editor

The Pattern editor provides specialized features for editing a pattern model.

The editor includes the following features:

  • Multiple pages editor : one page per concept
  • Names validation : pattern, parameters, methods
  • Automatic generation of internal content
  • Links with external editors : templates and Java code

The only ways to invoke the Pattern editor is either by double-clicking an in-development pattern in the Factory Component Explorer view, or by double-clicking a deployed one in the Factory Components view.

The editor is also automatically invoked when a new pattern is created, using the New Pattern Wizard.
