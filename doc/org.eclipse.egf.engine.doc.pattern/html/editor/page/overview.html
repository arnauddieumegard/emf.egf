Overview Page

The overview page displays the pattern general information and links to the other pages of a pattern editor.

General Information section

Field Description Modifiable Default value
Name The pattern logical name. Yes The name as specified in the pattern creation wizard.
Library id The containing library full id. No Generated at library creation time.
Id The pattern short id. No Generated at pattern creation time.
Description A free text description. Should explain as much as possible the pattern goals and interactions. Yes Auto-generated description for pattern name

Pattern Content section

Links to the other pages.

  • The Production page defines the production parameters applicable to pattern production.
  • The Specification page defines the pattern parameters and the condition linking those parameters.
  • The Inheritance page defines the pattern parent.
  • The Implementation page defines the pattern operations and their templates.
