Inheritance Page

The inheritance page defines the pattern parent hierarchy.
A pattern can inherit from another one at the uppermost.

The parent selection widget is made of two lists.

  • The first one, on the left is displaying all the potential parents.
  • The second one, on the right is showing the selected one.

To choose a parent, select it in the left list, and click the > button.
Selecting a library disables both buttons.
No new pattern can be selected on the left viewer, and transfered to the right one.


To choose a new parent, remove currently selected one.
Click the selected pattern on the right viewer, and click the < button.

Important note:
Only deployed patterns are shown in the left tree.
