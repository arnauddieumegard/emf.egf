EGF Generated Pattern

Identifier:
org.eclipse.egf.pattern.generatedPattern

Since:
[Enter the first release in which this extension point appears.]

Description:
EGF generated pattern extension point. Used to define generated patterns. A generated pattern is defined by an id, a template code string and a template engine type value.

Configuration Markup:

<!ELEMENT extension (generatedPattern)+>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>


<!ELEMENT generatedPattern EMPTY>

<!ATTLIST generatedPattern

id                 CDATA #REQUIRED

templateCode       CDATA #REQUIRED

templateEngineType (jet) "jet"

>

  • id - The unique id of the pattern in its containing library. A comment on characters that are allowed is needed here.
  • templateCode - The pattern code resulting from pattern generation.
  • templateEngineType - An attribute to define the template engine used to generate this pattern. If omitted, then it is jet by default. The attribute value will be one of the following:
    jet - as JET template engine.
