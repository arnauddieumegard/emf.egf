Pattern production

Since:

Description:
Pattern production extension point. Allows to point libraries and patterns to produce for a selected asset project.

Configuration Markup:

<!ELEMENT extension (patternProductionRoot)>

<!ELEMENT libraryOrPatternReference EMPTY>

<!ATTLIST libraryOrPatternReference

id CDATA #REQUIRED

>

  • id - The full id of the library or the pattern to produce.
