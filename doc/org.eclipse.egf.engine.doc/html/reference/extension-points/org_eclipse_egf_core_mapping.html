EGF Mapping

Identifier:
org.eclipse.egf.core.mapping

Since:
[Enter the first release in which this extension point appears.]

Description:
Extension point to define a mapping.

Configuration Markup:

<!ELEMENT extension (mapping)+>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>


<!ELEMENT mapping EMPTY>

<!ATTLIST mapping

id              CDATA #REQUIRED

name            CDATA #REQUIRED

path            CDATA #REQUIRED

assetIdAsSource CDATA #REQUIRED

assetIdAsTarget CDATA #REQUIRED

>

  • id - Mapping identifier.
  • name - Mapping name.
  • path - The ecore2ecore file defining the mapping.
  • assetIdAsSource - The identifier of asset that hosts the domain used as input in this mapping.
  • assetIdAsTarget - The identifier of asset that hosts the domain used as output in this mapping.
