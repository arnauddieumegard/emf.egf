New Pattern Library Wizard

The New Pattern Library wizard helps you create a new pattern library in a factory component.
The way to invoke this wizard is to select either a 'Pattern Library' or the 'Pattern Viewpoint' element in the tree viewer displayed by Factory Component Explorer view.

The creation of a pattern library is a three step process :

  1. Specify the pattern library definition
  2. Define the pattern library production orchestration [Optional]
  3. Define the pattern library runtime orchestration [Optional]
