New Mapping Wizard

The New Mapping wizard helps you create a new mapping in a factory component.
The way to invoke this wizard is to select the 'Mapping Viewpoint' element in the tree viewer displayed by Factory Component Explorer view.

Mapping Options

Option Description Default
Id Enter an identifier to make unique this mapping. The entered id has to be 'human' readable because it is requested when using the mapping services in patterns implementation.
Name Enter a name to identify the mapping. The entered name is used to create the related Ecore2Ecore file. The entered name must match file path naming.
It should only use the [a-zA-Z_0-9] set of characters.
Source The factory component that provides the domain used as input in the mapping. Choice is only available among deployed factory components containing a domain.
Target The factory component that provides the domain used as output in the mapping. Choice is only available among deployed factory components containing a domain.
