New Task Factory Wizard

The New Task Factory wizard helps you create a new Task Factory in an asset configuration.
The way to invoke this wizard is to select the 'Asset Configuration' element in the tree viewer displayed by Factory Component Explorer view.

Task Factory Options

Option Description Default
Class name Use the 'Browse...' button to select a class that implements com.thalesgroup.mde.mdsofa.rootasset.task.ITaskFactory interface The provided class provides a task implementation that makes some artifacts (e.g generate java code, java doc, ...).
