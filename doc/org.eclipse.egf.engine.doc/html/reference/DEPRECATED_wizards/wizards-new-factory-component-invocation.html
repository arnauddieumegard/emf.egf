New Factory Component Invocation Wizard

The New Factory Component Invocation wizard helps you create a new factory component invocation in a production plan.
The way to invoke this wizard is to select the 'Production Plan' element in the tree viewer displayed by Factory Component Explorer view.

Factory Component Invocation Options

Option Description Default
Name Enter a name to identify the factory component invocation. A user-friendly name to describe the overall task performed by referenced factory component.
Referenced Factory component A reference to a factory component. Choice is available among factory components located in either the target platform or the workspace.
