New Contract Element Wizard

The New Contract Element wizard helps you create a new contract element in a factory component.
The way to invoke this wizard is to select the 'Contract' element in the tree viewer displayed by Factory Component Explorer view.

The creation of a contract element is a two step process :

  1. Specify the contract element definition
  2. Define the contract element nature
