New Context Element Wizard

The New Context Element wizard helps you create a new context element in a factory component.
The way to invoke this wizard is to select the context owned by either the 'Production Plan' or the 'Asset Configuration'element in the tree viewer displayed by Factory Component Explorer view.

The creation of a context element is a three step process :

  1. Select the context element definition
  2. Define the context element nature
  3. Define the context element value
