Edit Factory Component Wizard

The Edit Factory Component wizard helps you edit an existing factory component in the workspace.
The way to invoke this wizard is to select an existing factory component in the tree viewer displayed by either Factory Component Explorer view or Factory Components view ( modifications are not available).

Mapping Options

Option Description Default
Name Entered name should be a human readable value describing the purpose of this factory component Created factory components by the default creation template have the name set to the related plug-in identifier.
Description Enter a text that explains the purpose of the factory component. Not mandatory to complete the wizard.
