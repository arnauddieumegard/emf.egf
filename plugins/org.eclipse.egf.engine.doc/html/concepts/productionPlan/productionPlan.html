Production Plan

Definition

A production plan defines an orchestration of factory invocations.

Motivation

A production plan is a kind of orchestration that meets the need of ordering asset manufacturing. Orchestration with a production plan is simple because it is just a sequential set of factory invocations. Complex orchestration, for instance with synchronization points, is not addressed here.

Structure

A production plan contains factory invocations, which declared the generation steps and implicitly the assets to be produced.
