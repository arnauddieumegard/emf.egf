EGF Perspectives

The EGF development tools contribute the following perspectives to the workbench:

EGF Engine Perspective

A perspective designed for working with Factory component projects. It consists of an editor area and the following views:
  • Package Explorer
  • Hierarchy
  • Outline
  • Properties
  • Problems
  • Javadoc
  • Factory Component Explorer view
  • Factory Components view
