New Pattern Wizard

The New Pattern wizard helps you create a new pattern in a factory component.
The way to invoke this wizard is to select a 'Pattern Library' element in the tree viewer displayed by Factory Component Explorer view.

Pattern Options

Option Description Default
Name Enter a name to identify the pattern. The entered name must match Java Class naming.
It should only use the [a-zA-Z_0-9] set of characters. Must start with a non-digit character.
