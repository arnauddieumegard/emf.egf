New Domain Wizard

The New Domain wizard helps you create a new domain in a factory component.
The way to invoke this wizard is to select the 'Domain' element in the tree viewer displayed by Factory Component Explorer view.

Domain Options

Option Description Default
Create Domain To select an existing domain, uncheck this checkbox. none
Package Name Enter a package name for the new domain. none
Domain file path The relative file path to the Ecore file within the current factory component that hosts the created or selected domain. Default value is set to model/domain/domain.ecore when creating the related Ecore file.
