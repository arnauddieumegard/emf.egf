Contract Element definition

The definition wizard page allows you to specify contract element name and its nature.

Option Description Default
Key Enter a name used as a key. none
Description Enter a text that explains the purpose of the contract element. Not mandatory to complete the wizard.
Type selection The type selection choice controls the contract element nature. Choices are a Java Type or a reference to another contract element.
Default value is set to Java Type.
Mandatory The mandatory check box controls the contract element nature. Checked means a context element has to reference this contract element to be able to run the factory of the hosting factory component.
Default value is set to checked.
