New Factory Component Wizard

The New Factory Componentwizard helps you create a new factory component in the workbench.
It is available under File > New > Project... > EGF > Factory Component Project from the top level menu.

The creation of a factory component project is a three-step process:

  1. Define the factory component project structure
  2. Specify the factory component content
  3. Add template extensions (optional)
