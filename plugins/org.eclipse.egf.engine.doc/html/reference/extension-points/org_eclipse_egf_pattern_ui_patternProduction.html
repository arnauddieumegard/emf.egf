Pattern production

Identifier:
org.eclipse.egf.pattern.ui.patternProduction

Since:
[Enter the first release in which this extension point appears.]

Description:
EGF Pattern production extension point. Allows to point libraries and patterns to produce for a selected asset project.

Configuration Markup:

<!ELEMENT extension (patternProductionRoot)>

<!ATTLIST extension

point CDATA #REQUIRED

<!ELEMENT patternProductionRoot (libraryOrPatternReference)+>

<!ATTLIST patternProductionRoot

id CDATA #REQUIRED

>

<!ELEMENT libraryOrPatternReference EMPTY>

<!ATTLIST libraryOrPatternReference

id CDATA #REQUIRED

>

  • id - The full id of the library or the pattern to produce.
